# Vlap-cli

Vlap 프론트 엔드

- React-Native: 사용자 인터페이스 프레임워크


## Hello React Native

1. NodeJS를 설치합니다.

2. 명령 프롬프트 창에서 다음 명령어를 입력합니다.
<pre><code> $ npm install -g yarn
 $ yarn global add react-native-cli
</code></pre>
 
3. Android Studio를 설치합니다.

4. Android Studio에서 SDK Manager를 열어서 다음 SDK를 설치합니다.

  - SDK Platforms - API 23 Marshmellow

  - SDK Tools - Android SDK Build-Tools

5. SDK가 설치된 경로를 환경변수 Path에 추가해줍니다.

  - SDK가 설치된 경로는 SDK Manager에서 확인할 수 있습니다.

6. "SDK가 설치된 경로\platform-tools " 경로를 환경변수 Path에 추가해줍니다.

7. Android Studio에서 AVD Manager를 열어 새로운 가상 머신을 만듭니다. 생성 시 System Image를 Marshmellow로 설정합니다.

8. AVD Manager에서 만들어진 가상 머신을 실행시킵니다.

9. 명령 프롬프트 창에서 다음 명령어를 입력합니다.
<pre><code> $ react-native init AwesomeProject
 $ cd AwesomeProject
 $ react-native start
</code></pre>

10. 새로운 명령 프롬프트 창을 열어 다음 명령어를 입력합니다. 생성된 프로젝트 폴더 아래서 입력하면 됩니다.
 <pre><code>$ react-native run-android
</code></pre>
